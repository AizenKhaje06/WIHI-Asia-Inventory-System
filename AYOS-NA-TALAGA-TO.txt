🔧 HARD REFRESH 404 - AYOS NA TO!

PROBLEMA DATI:
===============
Pag nag-hard refresh ka sa /dashboard/packing-queue:
1. RouteGuard nag-check ng auth
2. Habang nag-check, WALANG lumalabas (null)
3. Next.js nakita walang content → 404 ERROR

SOLUSYON:
=========
Inilipat ko yung RouteGuard para HINDI na naka-wrap sa buong layout!

DATI (MALI):
-----------
<RouteGuard>
  <Sidebar />      ← Naghihintay din ng auth check
  <Navbar />       ← Naghihintay din ng auth check
  <Content />      ← Naghihintay din ng auth check
</RouteGuard>

NGAYON (TAMA):
-------------
<Sidebar />        ← Lumalabas AGAD!
<Navbar />         ← Lumalabas AGAD!
<RouteGuard>
  <Content />      ← Ito lang naghihintay
</RouteGuard>

RESULTA:
========
✅ Sidebar - lumalabas AGAD kahit nag-check pa ng auth
✅ Navbar - lumalabas AGAD kahit nag-check pa ng auth
✅ Content - may loading spinner habang nag-check
✅ WALANG 404 ERROR NA!

PAANO I-TEST:
=============
