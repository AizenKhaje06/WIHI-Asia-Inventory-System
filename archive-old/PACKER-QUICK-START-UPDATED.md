## 3 Ways to Pack Orders

### Method 1: Camera Scanning (If Camera Works)
```
1. Click "Scan Barcode" button
2. Allow camera access (one-time)
3. Point camera at barcode
4. Order details appear automatically
5. Click "Mark as Packed"
6. Confirm
7. Scanner reopens for next order
```

### Method 2: Manual Input (NEW! Works Now!)
```
1. Click "Scan Barcode" button
2. Click "Enter Manually"
3. Type waybill number
4. Press Enter or click Search
5. Order details appear
6. Click "Mark as Packed"
7. Confirm
```

### Method 3: USB Barcode Scanner (RECOMMENDED!)
```
1. Plug USB scanner into computer
2. Click "Scan Barcode" button
3. Click "Enter Manually"
4. Scan barcode with USB scanner
5. Waybill auto-types in field
6. Press Enter
7. Order details appear
8. Click "Mark as Packed"
9. Confirm
```

### Method 4: View Button (Temporary)
```
1. Find order in Packing Queue table
2. Click "View" button
3. Order details appear
4. Click "Mark as Packed"
5. Confirm
```

## Camera Not Working?

### Quick Fix:
```
1. Go to: chrome://settings/content/camera
2. Find localhost:3000 in "Not allowed" list
3. Remove it
4. Refresh page
5. Try camera again
```

### Or Use Manual Input:
```
1. Click "Scan Barcode"
2. Click "Enter Manually"
3. Type or scan with USB scanner
```

## Troubleshooting

### Camera shows "Access Denied"
→ Use manual input or USB scanner

### Order not found
→ Check if waybill is correct
→ Check if order is already packed

### Scanner not working
→ Refresh page
→ Try manual input
→ Check browser console (F12)

### USB scanner not typing
→ Click in input field first
→ Check USB connection
→ Try different USB port
