# 🎥 Paano Ayusin ang Camera Permission

## Problema
Hindi lumalabas yung permission prompt para sa camera. Diretso "Access Denied" agad.

## Solusyon 1: I-clear ang Chrome Settings (PINAKAMABILIS)

### Step 1: Clear Site Data
```
1. Buksan ang Chrome
2. Press F12 (Developer Tools)
3. Right-click sa Refresh button
4. Click "Empty Cache and Hard Reload"
```

### Step 2: Reset Camera Permission
```
1. Click yung lock icon (🔒) sa address bar
2. Click "Site settings"
3. Hanapin yung "Camera"
4. Palitan from "Block" to "Ask"
5. Refresh yung page
```

### Step 3: Check Chrome Camera Settings
```
1. Punta sa: chrome://settings/content/camera
2. Tingnan sa "Not allowed" section
3. Hanapin yung localhost:3000
4. Click trash icon para tanggalin
5. Refresh yung app
```

## Solusyon 2: Windows Camera Settings

```
1. Windows Settings (Win + I)
2. Privacy & Security → Camera
3. I-enable lahat ng camera settings
4. Restart browser
```

## Solusyon 3: Gumamit ng Manual Input (BAGONG FEATURE!)

### Pwede na ngayon mag-type ng waybill number:
```
1. Click "Scan Barcode"
2. Click "Enter Manually"
3. Type yung waybill number
4. Press Enter o click Search
```

### O gumamit ng USB Barcode Scanner:
```
1. Plug USB scanner sa computer
2. Click sa search box
3. Scan barcode
4. Automatic na mag-type yung waybill
5. Press Enter
```

## Solusyon 4: Bumili ng USB Barcode Scanner (RECOMMENDED!)
