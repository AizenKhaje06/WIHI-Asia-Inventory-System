## 3 Paraan ng Pag-pack

### Paraan 1: Camera (Kung gumagana)
```
1. Click "Scan Barcode"
2. Allow camera
3. Scan barcode
4. Mark as packed
```

### Paraan 2: Manual Input (BAGO! Gumagana na!)
```
1. Click "Scan Barcode"
2. Click "Enter Manually"
3. Type waybill number
4. Press Enter
5. Mark as packed
```

### Paraan 3: USB Scanner (RECOMMENDED!)
```
1. Plug USB scanner
2. Click "Scan Barcode"
3. Click "Enter Manually"
4. Scan with USB scanner
5. Auto-type yung waybill
6. Press Enter
7. Mark as packed
```
